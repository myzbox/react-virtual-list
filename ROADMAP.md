# Roadmap

This document outlines the planned enhancements and long-term direction for
`@myzbox/react-virtual-list`.

The project follows semantic versioning.

---

## v1.x – Stable Line

Focus: Stability, adoption, and incremental improvements.
No breaking changes will be introduced in the v1.x series.

### Planned

- Bug fixes and performance improvements
- Improved documentation and examples
- Minor API enhancements (non-breaking)
- Better edge-case handling for large datasets

---

## v1.1.0 – Feature Enhancements

- Scroll-to-index API
- Expose visible range (`startIndex`, `endIndex`)
- Optional callback hooks (onScroll, onRangeChange)

---

## v2.0.0 – Major Update (Breaking Changes)

Focus: Advanced virtualization features.

### Planned

- Variable-height row support
- Dynamic size measurement and caching
- Improved rendering strategy for complex layouts

⚠️ v2.0.0 will introduce breaking changes.

---

## Long-Term Ideas

- Infinite loading helper
- Virtualized grid support
- Better SSR compatibility
- Accessibility improvements
